# MERN-Stack
🌐 HTML Practice Project

This repository contains a collection of basic HTML projects and practice files. It covers fundamental concepts of HTML including forms, tables, lists, text formatting, and building simple web pages.

📁 Project Structure
🔹 Main Files
index.html – Basic "Hello World" webpage
headtag.html – Demonstrates HTML document structure (head & body)
🔹 HTML Concepts Covered
📝 Forms
Form.html – Demonstrates input fields like:
Text, password, email, number
Radio buttons
Date picker


📋 Lists
lists.html – Includes:
Ordered lists
Unordered lists
Description lists
Nested lists


📊 Tables
Tables.html – Shows:
Table structure (thead, tbody, tfoot)
Column styling using <colgroup>


🔤 Text Formatting
textFormetters.html – Demonstrates:
Headings (h1–h6)
Paragraphs


🖼️ Images & Content Page
Images.html / onePiece.html – Includes:
Image usage
Text formatting with <sup> and <sub>
Structured content page (example: One Piece article)


📄 Resume Page
dummy_Resume.html – A sample resume webpage with:
Profile image
Education table
Projects section
Contact details


🚀 What You’ll Learn
Basic HTML structure
Working with forms and inputs
Creating tables and lists
Formatting text and content
Adding images to webpages
Building simple real-world pages (like resume)
🛠️ How to Run
Download or clone the repository
Open any .html file in your browser
Start exploring the output
